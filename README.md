# [เมนูวันนี้จากวัตถุดิบที่มี - What's in My Kitchen?]

ลิสต์เมนูอาหารที่แนะนำให้กับวัตถุดิบที่มีอยู่แล้วแต่คิดไม่ออกว่าจะทำเมนูอะไรดี ให้สำหรับคนที่มีวัตถุดิบแต่ไม่รู้จะทำเมนูอะไรดีให้เป็นตัวช่วยในการคิดเมนูอาหาร

## 🚀 ฟีเจอร์หลัก (Features)
- ฟังก์ชันเมนูอาหารตามวัตถุดิบที่มี

## 👥 สมาชิกและภาระงาน (Team Members & Responsibilities)
สมาชิกที่1 :นางสาวจิรภิญญา ชัยชมภู

| สัปดาห์ที่ (Week) | [สมาชิกคนที่ 1] | 
| :--- | :--- |

| **Week 2** | เตรียมโค้ดทดลองเบื้องต้น |
| **Week 3** | แก้ไขเว็ปแอพที่มีปัญหา |
| **Week 4** | ทดสอบระบบ (Testing) |
