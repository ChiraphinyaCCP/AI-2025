# 🍳 Cooking Time! - AI Personalized Nutrition
> **Project Concept:** ระบบแนะนำเมนูอาหารอัจฉริยะตามวัตถุดิบและโภชนาการรายบุคคล (Individual Project)
 
โปรเจกต์นี้พัฒนาขึ้นเพื่อแก้ไขปัญหาการเลือกเมนูอาหารในชีวิตประจำวัน มีฟังก์ชันสำหรับกรอกวัตถุดิบที่มีอยู่แล้วเพื่อหาเมนูที่เหมาะกับวัตถุดิบ และมีนำหลักการทางโภชนาการ (Nutrition Science) มาประยุกต์ร่วมกับการจัดการข้อมูล เพื่อสร้างระบบแนะนำอาหารที่เหมาะสมกับสุขภาพของผู้ใช้แต่ละคน

---
 
## 🚀 ฟีเจอร์หลัก (Key Features)
* **Smart Ingredient Matcher:** ค้นหาเมนูอาหารจากวัตถุดิบที่มี
* **Personalized Nutrition Analysis:** คำนวณค่า BMR และ TDEE อัตโนมัติ เพื่อวิเคราะห์โภชนาการรายบุคคล
* **Real-time Nutritional Visualization:** แสดงสัดส่วนสารอาหารมื้อนั้นด้วยกราฟวงกลม (Pie Chart)
* **Smart Scoring System:** ระบบให้คะแนนความเหมาะสมของเมนูอาหารตามเป้าหมายสุขภาพของผู้ใช้
 


 ## 🧠 AI Implementation :ส่วนที่เป็น AI ในโปรเจกต์
ในโปรเจกต์นี้มีการใช้ AI ในรูปแบบ **Expert System (ระบบผู้เชี่ยวชาญ)** และ **Decision Making Algorithm**:
* **Personalized Scoring:** อัลกอริทึมคำนวณความเหมาะสม (Matching Score %) โดยวิเคราะห์ส่วนเบี่ยงเบนของสารอาหารในเมนูเทียบกับความต้องการจริงของผู้ใช้ (BMR/TDEE)
* **Intelligent Recommendation:** ระบบตัดสินใจเลือกเมนู "Best Fit" ตามเป้าหมายสุขภาพที่ผู้ใช้กำหนด (เช่น Low Carb หรือ High Protein) อัตโนมัติ
* **Heuristic Logic:** การประมวลผลเงื่อนไขที่ซับซ้อนระหว่าง ข้อมูลส่วนบุคคล (น้ำหนัก/ส่วนสูง/กิจกรรม) และ ฐานข้อมูลอาหาร (Nutrition Database)

BMR (Basal Metabolic Rate) - อัตราเผาผลาญพื้นฐาน
นิยาม: พลังงานที่ร่างกายต้องการเพื่อคงชีวิต
ความสำคัญ: เป็นพลังงานที่ใช้ แม้คุณจะนอนเฉยๆ ตลอด 24 ชั่วโมง
การคำนวณ :
ผู้ชาย: 66 + (13.7 x น้ำหนักตัวเป็น กก.) + (5 x ส่วนสูงเป็น ซม.) – (6.8 x อายุ)
ผู้หญิง: 66.5 + (9.6 x น้ำหนักตัวเป็น กก.) + (1.8 x ส่วนสูงเป็น ซม.) – (4.7 x อายุ)

TDEE (Total Daily Energy Expenditure) - พลังงานรวมที่ใช้ต่อวัน

ลดน้ำหนัก: กินแคลอรี่ต่ำกว่า TDEE ประมาณ 300-500 kcal (หรือลดลง 20%)
รักษาน้ำหนัก: กินแคลอรี่เท่ากับ TDEE
เพิ่มน้ำหนัก/กล้ามเนื้อ: กินแคลอรี่มากกว่า TDEE
